## 🚀 Getting Started

the-bridge helps you hand off tasks to your own AI tools using normal files on your computer. You do not need to use APIs or set up a complex service. You place files in a folder, and the app reads and manages them.

This guide shows you how to download and run the app on Windows.

## 🪟 Run on Windows

1. Open the folder where you saved the download.
2. If the file is a ZIP, right-click it and choose Extract All.
3. Open the extracted folder.
4. Find the app file, such as `the-bridge.exe`.
5. Double-click the file to start the app.

If Windows asks for permission, choose Run anyway or Yes if you trust the download source.

## 📁 How it works

the-bridge uses a folder on your computer as the handoff point between you and your AI subscription. You write a task in a file, and the app helps move that task into the right place for your own AI setup.

A common flow looks like this:

1. Create a task file in the input folder.
2. Add the task you want handled.
3. Let the app pick up the file.
4. Read the result from the output folder.

This makes task delegation feel like normal file work instead of a separate online tool.

## 🛠️ First-time setup

1. Download the release from the release page.
2. Extract the files if needed.
3. Start the app.
4. Choose or confirm the folder it should watch.
5. Create your first task file in that folder.

A task file can be a plain text file. Keep the content short and clear. For example:

- Draft a reply to this email
- Turn these notes into a checklist
- Rewrite this text in plain English
- Summarize the meeting notes

## 🧾 Example folder flow

You can keep a simple folder layout like this:

- `Inbox` for new tasks
- `Working` for tasks in progress
- `Done` for finished tasks
- `Errors` for files that need attention

This setup keeps things easy to follow, even if you are not used to file-based tools.

## ✍️ Writing good task files

For best results, use one task per file. Give each file a clear name, such as:

- `email-draft.txt`
- `project-summary.txt`
- `todo-list.txt`

Inside the file, write:

- what you want done
- any limits or rules
- the format you want back
- any names, dates, or details that matter

Example:

Please turn these meeting notes into a short project update. Keep it under 150 words and use simple language.

## 🔍 Troubleshooting

If the app does not start:

1. Make sure you extracted the ZIP file first
2. Check that you downloaded the Windows release
3. Try running the file again
4. Move the app to a simple folder like `Downloads` or `Desktop`

If your task file is not processed:

1. Check that it is in the correct input folder
2. Make sure the file is plain text
3. Confirm the file name is easy to read
4. Make sure the app is running

If you do not see output:

1. Open the output folder
2. Look for a file with the same name or a related result file
3. Check for typos in the task text
4. Try a smaller task first
